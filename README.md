# NATO Learning Academy

Evaluation exercise - Bootstrap 5.2 boilerplate with best practices.

## Project Structure

```
NATO_Learning_Academi/
├── index.html              # Main HTML file
├── assets/
│   ├── css/
│   │   └── style.css       # Custom styles
│   ├── js/
│   │   └── main.js         # Custom JavaScript
│   └── images/             # Image assets
├── .gitignore              # Git ignore rules
└── README.md               # Project documentation
```

## Features

- **Bootstrap 5.2.3** - Latest stable version via CDN
- **Responsive Design** - Mobile-first approach
- **Best Practices** - Clean folder structure and organized code
- **Custom Styles** - CSS variables and utility classes
- **Modern JavaScript** - ES6+ with proper initialization
- **Accessibility** - Semantic HTML and ARIA considerations

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No build process required - works directly with CDN links

## Technologies Used

- HTML5
- CSS3 (with CSS Custom Properties)
- JavaScript (ES6+)
- Bootstrap 5.2.3

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Customization

### CSS Variables

Edit `assets/css/style.css` to customize:
- Color scheme
- Spacing values
- Typography settings

### JavaScript

Edit `assets/js/main.js` to add:
- Custom functionality
- Event handlers
- Component initialization
